## 系统要求

* Ruby 2.0.0 (or 1.9.3)
* PostgreSQL (or MySQL, SQLite)
* ImageMagick

## 安装步骤

```bash
git clone git://github.com/19wu/19wu.git
cd 19wu
bundle install --without sqlite3 mysql2
rake setup
rails server
```
